## Usage

Client command: `python3 client.py -s <SERVER_IP> -p <SERVER_PORT> -z <SOCKET_SIZE> -t <HASHTAG>`


Server Initialization: `python3 server.py -p <SERVER_PORT> -z <SOCKET_SIZE> -b <BACKLOG_SIZE>`
